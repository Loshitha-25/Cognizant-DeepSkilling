# Exercise 1 - Control Structures

## Scenario 1

Apply a 1% discount to loan interest rates for customers above 60 years of age.

## Scenario 2

Update VIP status for customers whose account balance is greater than $10,000.

## Scenario 3

Display reminder messages for customers whose loan due date is within the next 30 days.

## Tables Used

1. Customers
2. Loans

## PL/SQL Concepts Used

- FOR LOOP
- IF Condition
- UPDATE Statement
- DBMS_OUTPUT.PUT_LINE
- CURSOR FOR LOOP
